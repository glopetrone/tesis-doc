\section{Introducción}

% En el contexto de las redes neuronales y los modelos de aprendizaje automático, para determinar que tan bueno el modelo resultante es luego de entrenamiento existen muchas formas de evaluarlo. Estos métodos se suelen basar en primero entrenar el modelo y luego correrlo con un nuevo conjunto de datos no visto en entrenamiento y de ahi evaluar que tan bien generaliza para resolver nuestro problema. 

% Si el problema a resolver no involucra solo ese modelo, sino que un sistema con mas componentes que en conjunto se enfocan en resolver el problema, no es posible realizar esta forma de evaluación. El objetivo de este trabajo es plantear un método para poder evaluar un modelo en particular dentro de un sistema más grande, sin la necesidad de evaluarlo en conjunto con todo el sistema. En particular para los sistemas que buscan resolver un problema de clasificación binaria.
